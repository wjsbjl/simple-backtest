# 简易回测系统 simple-backtest
本文档使用向量化运算对已有因子值进行简单回测，通过初始因子值进行投资，并输出相关统计指标，这主要分为四部分。

## Step0 原始信号准备
这部分我们将gt(sin(close),log(bar))作为回测因子，其中：
 - close表示当前时间段收盘价。
 - bar表示当前时间段处于开盘后的第几个区间内（例如，此处采用分钟频率数据，9:30为开盘时间，则9:31的bar值为2）。
 - gt为逻辑判断函数，对于gt(x,y)，若x>y则取1，否则取0。

至于为何采用该因子，我们将在遗传算法生成因子的文档进行讨论。
## Step1 计算仓位
第一部分是sgnl2pos，在这里我们将原始信号处理为仓位信号，其中因子信号为正的判定为做多一手，因子信号为负的判定为做空一手，每日收盘时进行平仓。
## Step2 计算账户价值
第二部分是pos2value，这部分通过仓位信号生成投资后账户价值。其中手续费的考虑为先卖后买记作平今仓，手续费3.45/10000，其他情况下手续费为0.23/10000
## Step3 计算统计量
第三部分是value2sttc，这部分主要生成两种统计量，其中sttc_short输出收益回撤比率，sttc_long输出交易细节（包括每笔收益等）和收益统计量。
## Step4 绘制统计图表
第四部分是sttc2plot，这部分根据上一步生成的详细交易过程sttc_long绘制基本的统计图表，方便结果展现。

## 版本说明
本文档中信号值采取手动计算。
